# Modo Nova Feature

Checklist obrigatório antes de implementar qualquer feature nova:

1. **Verificar se já existe algo similar** no projeto antes de criar qualquer coisa
2. **Identificar todos os arquivos** que serão criados e modificados
3. **Confirmar padrão de pasta** com base nas features existentes — não criar arquivos fora do padrão estabelecido
4. **Listar dependências necessárias** — não adicionar pacotes sem justificativa
5. **Propor o plano completo** e aguardar aprovação antes de executar qualquer alteração
6. **Não começar a implementar** sem plano aprovado
