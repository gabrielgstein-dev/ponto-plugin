# Senior Ponto — Extensão Chrome

Extensão para calcular automaticamente seus horários de batimento de ponto na plataforma Senior.

## Como Instalar

1. Abra o Chrome e acesse: `chrome://extensions`
2. Ative o **Modo do desenvolvedor** (canto superior direito)
3. Clique em **"Carregar sem compactação"**
4. Selecione a pasta `senior-ponto-plugin`
5. A extensão aparecerá na barra do Chrome!

## Como Usar

### Modo Manual
1. Clique no ícone da extensão
2. Insira seu horário de **Entrada**
3. Insira o horário de **Almoço** quando for
4. Insira o horário de **Volta** quando retornar
5. A **Saída** é calculada automaticamente!

### Auto-Detecção
1. Esteja com a aba da plataforma Senior aberta
2. Clique no ícone da extensão
3. Clique em **"Detectar da página Senior"**
4. Os batimentos registrados serão importados automaticamente

### Widget Flutuante
- Um pequeno ícone de relógio aparece no canto inferior direito da página Senior
- Clique nele para ver seus horários sem abrir o popup

## Configurações
- **Jornada diária**: padrão 8 horas
- **Mínimo antes do almoço**: almoço disponível a partir de X horas trabalhadas
- **Duração do almoço**: tempo padrão do intervalo
- **Antecedência das notificações**: quantos minutos antes avisar

## Notificações
A extensão envia notificações automáticas:
- Quando está na hora do almoço
- Para voltar do almoço
- Quando está perto da hora de sair
- Quando a jornada foi cumprida
